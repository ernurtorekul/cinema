# Books & Reference Materials

This folder contains production books, guides, and reference materials used for:
- Cinematography rules
- Animation principles
- Camera work and lighting
- Scene composition
- Sound design guidelines
- Video editing techniques

## Usage

Place PDF or TXT files in this folder. The Source Material Agent will read these files and apply the rules to your video generation.

## Supported File Types
- `.pdf` - PDF documents
- `.txt` - Plain text files

## Example Content Structure

```
CINEMATOGRAPHY RULES
- Scene 1: Use wide establishing shot
- Tension scenes: Low angle, slow push-in
- Discovery moments: Close-up, rack focus

CAMERA MOVEMENTS
- Pan: Horizontal camera movement
- Tilt: Vertical camera movement
- Dolly: Camera moves toward/away from subject
- Truck: Camera moves sideways

LIGHTING
- Key light: Main light source
- Fill light: Softens shadows
- Back light: Separates subject from background
```
