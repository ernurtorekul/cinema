# Characters Folder

This folder contains character definitions for your video projects.

## How to use

1. Create a JSON file for each project or group of characters
2. Define characters with their traits, expressions, and poses

## Example character file format

```json
{
  "characters": [
    {
      "name": "Warrior",
      "traits": ["brave", "determined", "skilled fighter"],
      "style": "worn leather armor, scar on left cheek",
      "expressions": ["focused", "determined", "fierce", "weary"],
      "poses": ["battle stance", "crouching", "sword drawn", "kneeling"]
    },
    {
      "name": "Mystic",
      "traits": ["wise", "mysterious", "ancient knowledge"],
      "style": "flowing robes, glowing amulet, long white hair",
      "expressions": ["enigmatic", "thoughtful", "intense", "serene"],
      "poses": ["meditating", "hand raised casting spell", "standing with staff"]
    }
  ]
}
```

## File naming

Use descriptive names like:
- `fantasy-warriors.json`
- `sci-fi-crew.json`
- `commercial-actors.json`
